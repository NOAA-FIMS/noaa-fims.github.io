Probability mass function of the multinomial distribution.

\[ \frac{n!}{x_{1}! \dots x_{K}!}p_{1}^{x_{1}} \dots p_{K}^{x_{K}} \text{, } x_{i} \in \{0,...,n\}, i \in \{1,...,K\}, \text{ with } \sum_{i}x_{i} = n \text{ and } \sum^{K}_{k=1}p_{k}=1 \]
